== Description ==

A simple text slideshow block for reviews or such. Uses Splide.js.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/review-slideshow` directory.
2. Install Splide.js package: npm install @splidejs/splide
3. Activate the plugin through the 'Plugins' screen in WordPress.
